## 🌟 Featured Projects

I believe in building products that are **useful, reliable, and profitable**. Here are a few select projects representing my end-to-end design & delivery capabilities:

### 🛍️ [E-Commerce Suite](https://github.com/wemmyneat)
> **Stack:** Next.js (App Router), Django, PostgreSQL, Stripe
- Scalable, high-performance storefront designed with Tailwind CSS and shadcn/ui.
- Robust Django-powered inventory, checkout pipeline, and custom admin analytics dashboard.
- Includes smooth user journey optimization, SEO, and full-stack performance tuning.

### 📦 [Logistics Tracker App](https://github.com/wemmyneat)
> **Stack:** Flutter (Riverpod), Django REST Framework, Firebase Cloud Messaging
- Cross-platform Android & iOS tracking app using Flutter for fluid UI and state management.
- Real-time location polling, dispatch updates, push notification streams, and live driver tracking.
- Secured using robust REST APIs and custom JWT authentication.

### 📅 [Business Booking Platform](https://github.com/wemmyneat)
> **Stack:** Next.js, Django, PostgreSQL, Celery, Redis
- End-to-end booking and automated scheduling system reducing manual coordination.
- Automated email/SMS notification queues using Celery workers and Redis.
- Full responsive interface optimized for desktop and mobile devices.
